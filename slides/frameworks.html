Frameworks

CSS Frameworks

Some popular CSS Frameworks

  • Bootstrap: Mobile first, using Saas Or Lees, full set of components, supported by Twitter
  • Foundation: Mobile first, semantic, using Sass, supported by Zurb
  • Pure: Responsive, very small footprint, supported by Yahoo

kitchen Sink

Some advices

  • It's a great base
  • Don't import everything, you probably not use everything
  • Use themes (like stylebootstrap.info or bootswatch.com)
  • Create your own style keeping those key principles (semantic, mobile first, component, etc)

Font Frameworks

Why using a Font framework?

  • Smaller footprint than binary images
  • It's vectorial
  • Can apply CSS directives on it

  • It's monochrome

Some popular Font Frameworks

  • Glyphicons: Not free but partially included for free in Bootstrap
  • Font Icons: From Foundation
  • Font Awesome: Standalone, free, very complete, composition, etc.
