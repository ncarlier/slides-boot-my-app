CSS preprocessors

Extends the CSS language to decrease css development time and make the CSS more maintainable, themable and extendable.

Some popular preprocessors

  • Less: the Javascript based preprocessor
  • Sass: the Ruby based preprocessor (very powerfull with Compass)
  • Turbine: the PHP based preprocessor
  • and other like DtCSS (bad french joke inside)

Less

  • Preprocessor written in Javascript
  • Open source under Apache 2 Licence hosted on GitHub

Less features

  • Variables (with scope)
  • Mixins
  • Nested rules
  • Operations
  • Functions
  • Namespaces & Accessors
  • and other powerful stuff...

Preprocessor Pros and Cons

  • Definitely a great tool to improve CSS development!
  • Easy coupling with the build proccess.
  • Introduce the library concept

  • Not compatible with upcoming CSS3 specifications. (ex: var)
