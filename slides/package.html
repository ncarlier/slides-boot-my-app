Optimization of build process

  • Concat images: create sprites (grunt-montage)
  • Concat JS and CSS (grunt-usemin)
  • Minify JS and CSS (grunt-contrib-uglify and grunt-contrib-mincss)
  • Minify HTML (grunt-htmlcompressor)
  • Optimize image compression (grunt-contrib-imagemin)
  • etc.

Optimization tips

  • Serve the right resource: see the upcoming <picture> HTML5 tag
  • Use optimized web fonts
  • Use only what you need: icnfnt.com
  • Google SPDY module
  • Content Delivery Network (CDN)
  • Google page speed module (for lazy guys)
