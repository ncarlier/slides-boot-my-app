Scafolding

Generate code to scafold the application and fit Frameworks together.

Some popular scafolding tools:

  • Yeoman: http://yeoman.io
  • Brunch: http://brunch.io
  • Loom: https://github.com/rpflorence/loom

Yeoman

  • Command line interface written in Node.js
  • Open source under BSD Licence hosted on GitHub
  • Owned and supported by Google
  • Based on Yo, Grunt and Bower
  • Using concept of "generator" (inspired by RoR)
  • Over 680 generators!

Yeoman : Popular generators

  • angular: AngularJS
  • ember: Ember.js
  • backbone: backbone.js
  • jhipster: Full stack Framework (Yeoman + Maven + Spring + AngularJS) by Ippon Tech.
  • cordova: Apache Cordova is a platform for building native mobile applications using HTML5.
  • ionic: Build hybrid mobile apps using the Ionic Framework
  • reveal: Reval.js a great tool used to create this great presentation!
  • many more and maybe your own...

Scafolding Pros and Cons

  • Great tool to start from nothing!
  • Using a bunch of best practices.
  • Can be extended.

  • Usage generally limited to the bootstrapping.
  • Don't go outside the box.
  • It's not always the way i want to do it.
